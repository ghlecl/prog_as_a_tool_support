Profil

Profil de dose

Dose en fonction de la position latérale dans un faisceau de rayonnement.

Technique de mesure

  • On utilise un rail motorisé sur lequel le détecteur est posé.
  •  
  • À un intervalle de temps régulier, on prend une mesure de dose.
  •  
  • Le rail rapporte sa position lors de la mesure ce qui nous donne l'information de dose et de position.

Symétrie : axe central

  • Position 0 du détecteur centrée dans le champ implique axe du champ à la position 0.
  • Dans la réalité, il y a une incertitude sur la position du détecteur et le centre peut être décalé de quelques millimètres.
  • À tenir en compte lorsque l'on cherche la dose à l'axe dans nos implémentations.  Par contre, faible gradient.

Symétrie : 80% central

  • Une convention courante pour la dimension de champ est d'utiliser la distance entre les deux points dont la dose est 50% de celle à l'axe.
  • Pas nécessairement de donnée à 50% de la dose : interpolation nécessaire par l'implémentation.
  • Pourquoi pas toute la largeur : fort gradient dans les pénombres → petite erreur position = grande erreur dose.

Symétrie : équidistant

  • Les imperfections mécaniques des moteurs introduisent des incertitudes sur les positions.
  • Les données ne contiennent donc pas directement l'information recherchée.
  • À nouveau, une interpolation sera nécessaire.

Dose à l'axe

  • Les approximations sont les suivantes:
    • les profils sont déjà raisonnablement centrés;
    • les profils sont pratiquement constants autour de l'axe central.
  • Alors, on peut, sans faire une grande erreur, estimer que la dose à l'axe central est celle au point le plus près de 0.0.

Dose à l'axe

  • Le point le plus près de 0.0 est celui dont la valeur absolue est la plus petite.
  • Je me sert d'une array formula pour calculer une plage en mémoire contenant la valeur absolue des positions, puis je demande le minimum de cette place.
{=MIN(ABS(INDIRECT(H6):INDIRECT(L6)))}

Dose à l'axe

  • Ensuite, je recalcule le même array temporaire de valeur absolue et je compare chacune avec celle que j'ai trouvé.
  • Il n'y aura qu'une valeur vraie et j'utilise MATCH pour déterminer sa position.
{=MATCH(TRUE,ABS(INDIRECT(H6):INDIRECT(L6))=Q6,0)+F6-1}

Position des 50%

  • On a maintenant la dose à 100%.  On divise par deux, puis on doit chercher la position de cette dose.
  • Interpolation sera nécessaire puisqu'il se peut que la valeur ne soit pas directement dans les données.
  • Pour interpoler dans Excel, il faut générer les adresses des deux points autour de la valeur recherchée, puis utiliser l'équation d'une droite.

Comportement de MATCH

  • MATCH a un paramètre match_type qui détermine son comportement:
    • 1 → dernière valeur <= à lookup_value, tri croissant
    • 0 → première valeur == à lookup_value, aucun tri
    • -1 → dernière valeur >= à lookup_value, tri décroissant
  • Comme nous n'avons pas de garantie que le 50% de dose se retrouve dans le profil, nous devons utiliser 1 ou -1, mais pas 0.

Sécurité

  • Fonction de recherche "fuzzy" de MATCH que pour valeurs croissantes OU décroissantes, pas les deux.
  • Or, pour la dose d'un profil, on a les deux.
  • Position du zéro trouvée précédemment utilisée comme séparation pour assurer bon fonctionnement.

2. Position des 50%

half_axis_dose = axis_dose / 2.0

posi_50_ascend = np.interp( half_axis_dose,
                            data[:axis_idx,1],
                            data[:axis_idx,0]
                 )

posi_50_descend = np.interp( half_axis_dose,
                             data[:axis_idx:-1,1],
                             data[:axis_idx:-1,0]
                  )
  • Maintenant que l'on a la dose à l'axe, il suffit d'utiliser la fonction d'interpolation de Numpy pour déterminer la dose au 50% pour les deux moitiés du profil.

2. Position des 50%

# np.interp( nouvelle_valeur_x, données_x, données_y )
#
# indexation[ a : b : c ] → a est le début,
#                           b est la fin,
#                           c est le pas
#      défaults de a et b → si c > 0 : a = 0, b = len - 1
#                           si c < 0 : a = len - 1, b = 1 avant début

posi_50_ascend = np.interp( half_axis_dose,
                            data[:axis_idx,1],
                            data[:axis_idx,0]
                 )

posi_50_descend = np.interp( half_axis_dose,
                             data[:axis_idx:-1,1],
                             data[:axis_idx:-1,0]
                  )

3. Position du centre

width = posi_50_descend - posi_50_ascend

center = posi_50_descend - ( width / 2.0 )
  • Avec les deux positions, on peut déterminer la largeur et le centre.

4. Recaler le profil

data[:,0] -= posi_50_descend - ( width / 2.0 )
  • On peut maintenant appliquer cette correction de position au profil pour le recentrer.
  • Puisque la position du centre ne servira plus, il ne sert à rien d'en faire une variable explicite et on utilise directement l'expression ici.

5. Positions équidistantes

symetry_range = 0.8 * width # 80% of the beam width

step = 0.001

half_pos = np.arange( symetry_range / 2.0, 0.0 + step, -step )

lt_half_doses = np.interp( half_pos, data[:,0], data[:,1] )
rt_half_doses = np.interp( -half_pos, data[:,0], data[:,1] )
  • On génère des positions connues d'un côté du profil, on prend le négatif de ces positions pour l'autre côté.
  • On interpole la dose à toutes ces positions.

6. Valeurs absolue diff

np.abs( lt_half_doses - rt_half_doses )
  • Suffit d'appliquer la fonction np.abs à la différence des array de dose.

7. max / axe = symétrie

new_axis_dose = np.interp( 0.0, data[:,0], data[:,1] )
sym = np.max( np.abs( lt_half_doses - rt_half_doses ) )/new_axis_dose * 100.0
  • On trouve d'abord la dose à la position 0.0 par interpolation.
  • On trouve ensuite le max avec np.max et on divise.
  • Voilà, l'implémentation de l'algorithme est terminée.

Complet

# Read file content
data = np.genfromtxt( cur_file, skip_header=2 )

# Find approximate axis dose
axis_idx = np.argmin( np.abs( data[:,0] ) )
axis_dose = data[axis_idx, 1]

# Find (and apply) position shift based on position of 50% doses
half_axis_dose = axis_dose / 2.0
posi_50_ascend = np.interp( half_axis_dose, data[:axis_idx,1], data[:axis_idx,0] )
posi_50_descend = np.interp( half_axis_dose, data[:axis_idx:-1,1], data[:axis_idx:-1,0] )
width = posi_50_descend - posi_50_ascend
data[:,0] -= posi_50_descend - ( width / 2.0 )

# Create new position arrays for the left and right halves of the profile
symetry_range = 0.8 * width # 80% of the beam width
step = 0.001
half_pos = np.arange( symetry_range / 2.0, 0.0 + step, -step )

# Interpolate the doses and calculate the symmetry
lt_half_doses = np.interp( half_pos, data[:,0], data[:,1] )
rt_half_doses = np.interp( -half_pos, data[:,0], data[:,1] )
new_axis_dose = np.interp( 0.0, data[:,0], data[:,1] )
sym = np.max( np.abs( lt_half_doses - rt_half_doses ) )/new_axis_dose * 100.0

# Output the results
print( "(" + os.path.basename( cur_file ) + ")\tsymétrie: " +
                                    "{sym:.2f}".format( sym=sym ) )

def read_file( file_name : str ):
   data = []
   with open( file_name, 'r' ) as f:
      f.readline() # first header line
      f.readline() # second header line
      for line in f:
         vals = line.strip().split()
         data.append(
            [ float( vals[0].strip() ),
              float( vals[1].strip() ) ]
         )
   return data
  • Il ne reste plus qu'à amalgamer ces données dans une nouvelle liste et à la retourner pour compléter la fonction.
  • Pour l'utiliser, on a qu'à lui passer un path pour un de nos fichiers.
